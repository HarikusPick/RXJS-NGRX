# RxJS NgRx Lab

## EN

An educational Angular project built for developers who are just starting to learn Angular, RxJS, and NgRx.

I created this app to make concepts that often feel abstract or confusing easier to understand through small, practical examples. The project includes hands-on sections for RxJS operators, `Promise vs Observable`, `Signals`, `Subjects`, basic NgRx store structure, and more advanced NgRx patterns.

The goal is not to present a production-ready application, but to provide a learning playground where beginners can read the code, run the app, and connect concepts with visible behavior on the screen.

Topics you can explore in this project:

- Reactive thinking with RxJS
- Observable, Subject, and operator basics
- Differences between Promise and Observable
- Core Angular Signals usage
- NgRx Store, Actions, Reducers, Selectors, and Effects
- Cleaner and more modular state management patterns

If you are new to the Angular ecosystem, this repository can serve as both a reference project and a practice space.

Angular, RxJS ve NgRx konularini yeni ogrenmeye baslayanlar icin hazirlanmis egitim odakli bir ornek proje.

Bu uygulamayi, teoride karisik gelebilen kavramlari daha gorunur ve denenebilir hale getirmek icin gelistirdim. Projede RxJS operatorleri, `Promise vs Observable`, `Signals`, `Subjects`, temel NgRx store yapisi ve biraz daha ileri NgRx senaryolari adim adim incelenebilir.

Amacim, yeni baslayan birinin sadece kodu okumasi degil, ekranda sonucu gorerek mantigi daha kolay kavramasi. Bu yuzden proje bir "production app" olmaktan cok, pratik yapma ve mantik oturtma ortami olarak tasarlandi.

Bu projede ogrenebilecegin basliklar:

- RxJS ile reaktif dusunce yapisi
- Observable, Subject ve operator mantigi
- Promise ve Observable farklari
- Angular Signals temel kullanimi
- NgRx Store, Actions, Reducers, Selectors ve Effects
- Daha moduler ve okunabilir state management yaklasimlari

Eger Angular ekosistemine yeni giriyorsan, bu repo sana hem referans hem de deneme alani olarak yardimci olabilir.


## Features

- `products`: RxJS-based search flow
- `posts`: NgRx store example
- `operators`: RxJS operator demos
- `promise-vs-obs`: Promise and Observable comparison
- `signals`: Angular Signals examples
- `subjects`: Subject types and usage
- `ngrx-advanced`: advanced NgRx and ComponentStore examples

## Getting Started

```bash
npm install
npm start
```

Open `http://localhost:4200/` in your browser.
