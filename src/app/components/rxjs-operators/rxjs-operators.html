3 — Düzleştirme: switchMap / mergeMap / concatMap

Bir observable'ın içinden yeni observable döndürürken hangisini seç? Aynı 3 HTTP isteğini (post 1, 2, 3) üç farklı şekilde yönet ve sonuçları karşılaştır.

switchMap Yeni gelince öncekini iptal et 
// Kullanım: arama kutusu, route değişimi
// Önceki istek tamamlanmadan yeni gelirse
// önceki otomatik iptal edilir
of(1, 2, 3).pipe(
  switchMap(id => http.get('/posts/' + id))
)
@if (!switchMapOutput().length) { Butona tıklayın... } @for (line of switchMapOutput(); track $index) {

}
mergeMap Hepsini eş zamanlı çalıştır 
// Kullanım: paralel yüklemeler
// Tüm inner observable'lar aynı anda
// çalışır, sonuçlar karışık gelebilir
of(1, 2, 3).pipe(
  mergeMap(id => http.get('/posts/' + id))
)
@if (!mergeMapOutput().length) { Butona tıklayın... } @for (line of mergeMapOutput(); track $index) {
{{ line }}
}
concatMap Sırayla, biri bitmeden diğeri başlamaz 
// Kullanım: sıralı upload, adım adım akış
// Her inner tamamlanmadan diğeri
