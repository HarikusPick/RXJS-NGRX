Junior Temel Signal API'leri

signal() — Yazılabilir reaktif değer

Zone.js'e gerek kalmadan değişiklikleri takip eden Angular'ın temel reaktivite primitifi. .set() ile doğrudan ata, .update() ile önceki değere göre değiştir. 


    

      
      {{ count() }}
      
      
    

    
💡 Template'te count() ile okuyorsun. Angular, bu çağrıyı bir bağımlılık olarak kayıt eder.

  


  
  

    
computed() — Memoize edilmiş türetilmiş değer

    

      Bağımlı sinyaller değişmeden yeniden hesaplanmaz. Performanslı ve salt-okunur.
    

    

    

      

        {{ cartCount() }} ürün
        Toplam: {{ cartTotal() }}₺
        
      

      @for (item of cartItems(); track item.id) {
        

          {{ item.name }}
          {{ item.price }}₺
          
        

      }
    

    
💡 Ürün ekle/çıkar — cartTotal() ve cartCount() yalnızca cartItems() değişince güncellenir.

  


  
  

    
effect() — Reaktif yan etki

    

      Okuduğu sinyallerden biri değiştiğinde otomatik yeniden çalışır.
      DOM manipülasyonu, loglama, localStorage senkronizasyonu için kullanılır.
    

    

    

      
      
      
    

    

      @if (!effectLog().length) {
        Butonlara tıkla — effect() çalıştığında buraya yazar...
      }
      @for (line of effectLog(); track $index) {
        
{{ line }}

      }
    

    
💡 effect(), count VEYA theme değişince tetiklenir — iki bağımlılığı birden takip eder.

  


  
  

    Mid
    RxJS ↔ Signal köprüsü & bileşen I/O
  


  
  

    
toSignal() — Observable'ı Signal'a çevir

    

      async pipe'ın yerini alır. Otomatik abone olur, bileşen destroy olunca otomatik iptal eder.
      HTTP ve interval Observable'larını Signal'a dönüştür.
    

    

    

      

        interval(1000) → toSignal()
        {{ tickerSignal() }}
      

      

        HTTP GET → toSignal() — {{ topProducts().length }} ürün
        

          @for (p of topProducts(); track p.id) {
            {{ p.title }}
          }
        

      

    

    
💡 Template'te tickerSignal() — subscribe, unsubscribe veya async pipe yok. toSignal her şeyi yönetiyor.

  


  
  

    
toObservable() — Signal'ı Observable'a çevir

    

      Signal'a debounce, switchMap gibi RxJS operatörleri uygulamak istediğinde kullan.
      Signal değişimlerini Observable stream'e çevirir.
    

    

    
    @if (searchLoading()) {
      
Aranıyor...

    }
    

      @if (!searchLoading() && !searchResults().length) {
        Yazmaya başla → toObservable + switchMap çalışır...
      }
      @for (p of searchResults(); track p.id) {
        {{ p.title }}
      }
    

  


  
  

    
input() / output() — Signal tabanlı bileşen I/O

    

      @Input() ve @Output()'ın modern yeni hali.
      Child bileşen, input()'u computed() içinde doğrudan kullanabilir.
    

    

    

      @for (p of cardProducts(); track p.id) {
        
      }
      @if (!cardProducts().length) {
        Tüm kartlar kaldırıldı.
      }
    

    @if (removedLog().length) {
      

        @for (line of removedLog(); track $index) {
          
{{ line }}

        }
      

    }
    
💡 SignalCardComponent'te total = computed(() => basePrice() * quantity()) — input signal'ını computed içinde kullanıyor.

linkedSignal() — Türetilmiş ama yazılabilir signal

    

      computed() gibi bağımlılık takibi yapar, ama .set() / .update() ile
      manuel güncelleme de mümkündür. Sayfalama + filtre gibi durumlar için idealdir. (Angular 19+)
    

    

    

      

        @for (cat of categories; track cat) {
          
        }
      

      

        
        Sayfa {{ currentPage() }}
        
      

      
💡 Kategori değiştiğinde sayfa otomatik 1'e döner — linkedSignal bunu yapar. Sayfayı manuel artırabilirsin ama kategori değiştirince sıfırlanır.

    

  


  
  

    
rxResource() — Reaktif HTTP kaynağı

    

      Request sinyalleri değiştiğinde otomatik HTTP isteği atar. effect() + subscription yönetimi gerekmez. (Angular 19+)
    

    

    

      

        {{ selectedCategory() }} / sayfa {{ currentPage() }}
        @if (productsResource.isLoading()) {
          Yükleniyor...
        }
        
      

      @if (productsResource.isLoading()) {
        
Veriler çekiliyor...

      }
      @if (productsResource.error()) {
        
Hata oluştu.

      }
      

        @for (p of (productsResource.value() ?? []); track p.id) {
          

            
            {{ p.title }}
            {{ p.price }}$
          

        }
      

    

    
💡 Kategori veya sayfa değiştiğinde rxResource otomatik yeni istek atar. Effect, subscribe veya unsubscribe yazmadık.
